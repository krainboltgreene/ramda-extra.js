# ramda-startsWith

![Version][BADGE_VERSION]
![Tests][BADGE_TRAVIS]
![Stability][BADGE_STABILITY]
![Dependencies][BADGE_DEPENDENCY]

> string -> string -> boolean

Determines if a given subset of text is at the start of another set of text.

``` javascript
startsWith("Hello")(data)
```

Would return:

``` javascript
true
```
